# 👑 The King's Path — Em Busca da Coroa

> *Somente o mais habilidoso conseguirá chegar até a coroa e se tornar o verdadeiro rei.*

## 🎮 Sobre o Projeto

**The King's Path** é um jogo de labirinto desenvolvido no Scratch, onde o jogador deve navegar por um caminho cheio de mistérios e armadilhas, desviando dos espinhos para conquistar a coroa.

Este projeto foi desenvolvido como parte dos estudos de **Pensamento Computacional** na UNIVESP, inspirado no ciclo de aprendizagem de Mitchel Resnick:

> *"Imaginar → Criar → Brincar → Compartilhar → Refletir"*

## 🕹️ Como Jogar

- Use as teclas **W, A, S, D** ou as **setas direcionais** para movimentar o personagem
- ⚠️ Desvie dos **espinhos** — um passo em falso pode custar sua jornada!
- Chegue até a **coroa 👑** para vencer

🔗 **[Jogar no Scratch](https://scratch.mit.edu/projects/1211530685/)**

## 🛠️ Tecnologias

- [Scratch 3.0](https://scratch.mit.edu/) — MIT Media Lab
- Lógica de movimentação com eventos de teclado
- Detecção de colisão com obstáculos
- Design de labirinto e sprites personalizados

## 📚 Contexto de Aprendizado

Mais do que um jogo, este projeto foi um exercício prático de pensamento computacional:
- **Decomposição** — quebrar o problema em partes menores
- **Reconhecimento de padrões** — identificar o comportamento dos obstáculos
- **Abstração** — focar no que realmente importa para chegar ao objetivo
- **Algoritmos** — criar o caminho lógico até a coroa
